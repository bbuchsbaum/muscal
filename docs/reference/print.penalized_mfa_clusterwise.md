# Print Method for Penalized MFA Clusterwise Objects

Prints a summary of a `penalized_mfa_clusterwise` object, showing key
parameters (lambda, components), convergence info, and block details.

## Usage

``` r
# S3 method for class 'penalized_mfa_clusterwise'
print(x, ...)
```

## Arguments

- x:

  A `penalized_mfa_clusterwise` object (which inherits from
  `multiblock_projector`)

- ...:

  Additional parameters (unused)

## Value

Invisibly returns the input object
