# Plot Block Weights

Bar chart showing the normalization weights (alpha) for each block.

## Usage

``` r
plot_block_weights(x, ...)
```

## Arguments

- x:

  A fitted \`mfa\` or \`linked_mfa\` object.

- ...:

  Additional arguments passed to \[barplot()\] or ggplot.

## Value

A ggplot object or base R plot.
