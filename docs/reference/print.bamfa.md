# Print Method for BaMFA Objects

Prints a concise summary of a `bamfa` object fitted with the EM-like
algorithm, highlighting the model structure and fit.

## Usage

``` r
# S3 method for class 'bamfa'
print(x, ...)
```

## Arguments

- x:

  An object of class `bamfa`.

- ...:

  Additional parameters (unused).

## Value

Invisibly returns the input object.
