# Validate arguments for penalized MFA functions

Validate arguments for penalized MFA functions

## Usage

``` r
validate_pmfa_args(
  data_list,
  coords_list = NULL,
  ncomp,
  lambda,
  max_iter,
  nsteps_inner,
  learning_rate,
  optimizer,
  beta1,
  beta2,
  adam_epsilon,
  tol_obj,
  tol_inner,
  verbose,
  memory_budget_mb
)
```
